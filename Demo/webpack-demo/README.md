。

## How To Use?

#### Install

```bash
npm i
```

#### Run Project

```bash
npm start
```

#### Build Project

```bash
npm run build
```

## RequiredKit

- [x] webpack-cli
- [x] babel
- [x] webpack-dev-server
- [x] html-webpack-plugin
- [x] raw-loader
- [x] webpack.HotModuleReplacementPlugin
